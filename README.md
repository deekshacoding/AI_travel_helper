# 🌍 AI Travel Research Agent

An intelligent **AI-powered Travel Research Agent** built with **Python**, **Streamlit**, **LangChain**, **Google Gemini**, and the **Groq API**. This application helps users plan their trips effortlessly by researching destinations, recommending top-rated hotels, generating personalized travel itineraries, creating packing checklists, and providing an interactive AI travel assistant for answering travel-related questions.

---

## ✨ Features

* 🌎 **Destination Research**

  * Get detailed information about any destination.
  * Discover popular tourist attractions.
  * Learn about local culture, weather, food, transportation, and travel tips.

* 🏨 **Hotel Recommendations**

  * Suggests hotels with **4-star ratings and above**.
  * Provides multiple accommodation options for different budgets.

* 🗓️ **Smart Travel Itinerary Generator**

  * Automatically creates day-wise travel plans.
  * Optimized schedules for sightseeing and exploration.
  * Suitable for solo travelers, families, couples, and groups.

* 🎒 **Travel Necessities Checklist**

  * Personalized packing list.
  * Essential travel documents.
  * Clothing recommendations.
  * Health and safety items.
  * Electronics and accessories.
  * Destination-specific travel essentials.

* 🤖 AI Travel Assistant Chatbot

  * Ask unlimited travel-related questions.
  * Get instant AI-generated travel advice.
  * Receive recommendations during trip planning.
  * Interactive conversational interface.

* 📍 Place Recommendations

  * Famous tourist attractions.
  * Hidden gems.
  * Local experiences.
  * Food recommendations.
  * Shopping suggestions.
  * Best time to visit.

* ⚡ Fast AI Responses

  * Powered by **Groq API** for high-speed inference.
  * Uses **Google Gemini** for intelligent reasoning.
  * Managed with **LangChain** for prompt orchestration.

* 🎨 Modern Streamlit Interface

  * Clean and responsive UI.
  * Easy-to-use travel planner.
  * Interactive chatbot.
  * Organized travel reports.

# 🚀 Installation

## 1. Clone the Repository

```bash
git clone https://github.com/yourusername/travel-research-agent.git

cd travel-research-agent
```

---

## 2. Create Virtual Environment

Windows

```bash
python -m venv venv

venv\Scripts\activate
```

Mac/Linux

```bash
python3 -m venv venv

source venv/bin/activate
```

---

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4. Configure API Keys

Create a **.env** file inside the project directory.

```env
GROQ_API_KEY=your_groq_api_key

GOOGLE_API_KEY=your_gemini_api_key
```

---

## 5. Run the Application

```bash
streamlit run langchain_research.py
```

# 💡 How It Works

1. Enter the destination you want to visit.

3. It recommends highly-rated hotels.
4. Generates a complete travel itinerary.
5. Creates a travel necessities checklist.
6. Displays important travel information.
7. Chat with the AI assistant for additional travel advice.
